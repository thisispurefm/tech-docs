# PureNet

The internal network within the Studios

## Network Layout

Router - `10.0.0.1`  
Subnet - `10.0.0.0/16`  
Static Address Pool - `10.0.0.100` to `10.0.0.200`  
DHCP Address Pool - `10.0.100.1` to `10.0.100.254`
